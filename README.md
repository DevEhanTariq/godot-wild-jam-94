# Chemical XYZ

### Description:
- This game was made for the **Godot Wild Jam #94**!
- It was made in 6 out of the **9 days** given for the game jam.
- I over scoped the project, causing the theme, **Mutation**,  not to be implemented in the game.
- This was also my first game jam.
- The Game was made in **Godot v4.6.3**
- The [Itch.io](https://devehantariq.itch.io/chemical-xyz) link to the game

### Game Concept:
- It's a puzzle game that requires you, the *Chemist*, to put different Chemicals through different processes to achieve the desired Chemical.
